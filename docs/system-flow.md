# ระบบแจ้งซ่อม - Flow การทำงาน

## ภาพรวมระบบ

```
┌─────────────────────────────────────────────────────────────────┐
│                     ระบบแจ้งซ่อม (Maintenance System)              │
├─────────────────────────────────────────────────────────────────┤
│  User (ผู้แจ้ง)  →  Admin (ผู้ดูแล)  →  Technician (ช่าง)       │
└─────────────────────────────────────────────────────────────────┘
```

---

## 1. Flow หลัก - การแจ้งซ่อม

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│  ผู้แจ้ง   │────▶│   Admin  │────▶│   ช่าง   │────▶│  เสร็จสิ้น │
│  แจ้งซ่อม  │     │ มอบหมาย  │     │ ดำเนินการ │     │  แจ้งผู้แจ้ง│
└──────────┘     └──────────┘     └──────────┘     └──────────┘
     │                │                │                │
     │ 📧 Email       │ 📧 Email       │                │ 📧 Email
     │ แจ้ง Admin     │ แจ้งช่าง       │                │ แจ้งผู้แจ้ง
     ▼                ▼                ▼                ▼
```

## 2. สถานะงานซ่อม

```
  ┌─────────────┐
  │   pending   │  ← สร้างใหม่
  │ (รอดำเนินการ) │
  └──────┬──────┘
         │ Admin มอบหมายช่าง
         ▼
  ┌─────────────┐
  │ in_progress │
  │(กำลังดำเนินการ)│
  └──────┬──────┘
         │ ช่าง/Admin กดเสร็จ
         ▼
  ┌─────────────┐
  │  completed  │
  │  (เสร็จสิ้น)  │
  └─────────────┘

  * สามารถ "ยกเลิก" ได้ทุกขั้นตอน → status: cancelled
```

---

## 3. บทบาทผู้ใช้ (Roles)

### User (ผู้ใช้งาน)
- แจ้งซ่อมใหม่ + แนบรูป + เลือกวันที่
- ดูรายการแจ้งซ่อมของตัวเอง
- ดูสถานะงาน + รูปภาพ + วิธีการซ่อม

### Admin (ผู้ดูแลระบบ)
- ดู Dashboard + กราฟสถิติ
- ดูรายการแจ้งซ่อมทั้งหมด + กรอง + Export Excel
- มอบหมายงานให้ช่าง
- เปลี่ยนสถานะงาน
- เพิ่มรายละเอียดวิธีการซ่อม
- เพิ่มรูปภาพ
- ลบรายการ / ลบทั้งหมด
- จัดการพนักงาน (เพิ่ม/แก้ไข/ลบ/รีเซ็ทรหัสผ่าน)
- ตั้งค่าองค์กร (โลโก้/ชื่อ/หมวดหมู่/แผนก/Email)

### Technician (ช่าง)
- ดู Dashboard
- ดูเฉพาะงานที่ถูกมอบหมายให้ตัวเอง
- เปลี่ยนสถานะงาน (เริ่ม/เสร็จ/ยกเลิก)
- บันทึกวิธีการซ่อม
- เพิ่มรูปภาพ

                        ▼

    ┌──────────┐  ┌──────────┐
    │จัดการ     │  │ตั้งค่า    │   ← เฉพาะ Admin
    │พนักงาน   │  │องค์กร    │
    └──────────┘  └──────────┘

## 7. ข้อมูลที่เก็บ

| ตาราง | ข้อมูล |
|-------|--------|
| users | ผู้ใช้ (ชื่อ, แผนก, สิทธิ์, email, โทร) |
| repair_requests | งานแจ้งซ่อม (หัวข้อ, รายละเอียด, สถานะ, ผู้แจ้ง, ผู้รับผิดชอบ) |
| request_history | ประวัติการดำเนินการ |
| request_images | รูปภาพประกอบ (เก็บ path ไปยังไฟล์ใน uploads/) |
